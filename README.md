# Cloud DataProc cluster

Google Cloud DataProc cluster
build a security hardened, HA cluster capable of running Hadoop jobs

Leverage: 
- Google Cloud Platform, Data Proc Cluster:    
https://cloud.google.com/dataproc/

- FaceBook Presto:    
https://prestodb.io/

to deploy:    

1. gcloud init    
1. clone this repo    
1. CD to this repo   
1. run "make"
